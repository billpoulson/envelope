How to read this view

Each row is one variable. Layer columns run left → right (bottom of the stack → top).

  • Collapse Layer columns start collapsed (narrow strip with markers). Click ▶ in a header to show that layer’s values; ▼ collapses again. Bundle name stays visible when collapsed.
  • When a column is collapsed

    Gray — the key is also defined in a layer to the right (upper layer wins).

    Green — this layer defines the key and the next layer to the right does not (or this is the top layer).

  • Result The Merged export column shows the final value after applying the full stack.
  • Secrets Entries stored as secret in a bundle show as (secret) until you enable Show secret values or right-click and choose View secret…. Plaintext variables are always shown.
  • Edit Right-click a layer cell that has a value to open a menu and jump to Edit for that variable in the source bundle.
  • No value When a row shows red markers (no effective value in any layer), right-click a layer cell and choose Define value in this layer… to set the variable in that bundle.
